Quatro jogadores realizam um jogo de dados. O dado tem as faces 
codificadas com 0, 1, -2, 3, 4, -5. Primeiramente é sorteado o Jogador 
que irá Começar. Cada vez que um jogador realizar sua jogada, sorteará o 
número do próximo jogador. O vitorioso será o jogador que acumular mais pontos
depois de 10 jogadas. Imprimir o nome do vencedor e seus pontos.
